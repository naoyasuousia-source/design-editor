# HTMLデザインエディター プロジェクト設計書

このプロジェクトの目的は、テキストと画像が分離したHTMLデザインを効率的に作成・管理するためのWebアプリを構築することです。Antigravity（AI）とユーザーが協調してデザインをブラッシュアップしていくワークフローを実現します。

## 主な機能

1. **デザイン表示エリア (Design Canvas)**
   - Antigravityが生成したHTML（innerHTML）がリアルタイムで反映される領域です。
   - ユーザーはここで実際の見栄えを確認します。

2. **ファイル操作 (File System Access API)**
   - **新規保存**: 現在のデザインエリアの内容を、名前を付けてローカルのHTMLファイルとして保存します。
   - **ファイルを開く**: 保存済みのHTMLファイルを選択して読み込み、デザインエリアに展開します。
   - **上書き保存**: 読み込み中のファイルに対して、現在のデザインを上書き保存します。

3. **AI連携ワークフロー**
   - ユーザーはAntigravity（ワークスペース）に対して変更を依頼します。
   - Antigravityは、アプリ内の特定の要素の `innerHTML` を直接修正、あるいは修正案を提示します。
   - ユーザーはブラウザ上で反映された結果を確認し、必要に応じてローカルへ保存します。

## 技術スタック
- **Frontend**: HTML5, Vanilla CSS, TypeScript
- **Tooling**: Vite (Dev Server)
- **APIs**: File System Access API

## プロジェクト構造
- `index.html`: メインUI（エディター画面）
- `src/main.ts`: ファイル操作ロジック、デザインエリアの同期管理
- `src/style.css`: プレミアムなデザインシステム定義
